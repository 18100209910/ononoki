## 安装

### 在线安装（推荐）

[Chrome 商店](https://chrome.google.com/webstore/detail/meme-maker/hlalndcfbinfampnholjnkcaimdgnfae)

### 离线安装

下载 [crx 文件](./main.crx)，打开 chrome 扩展页面，把 crx 文件拖进去即可。

可参考下面的动图

![install](./how_to_install_offline_crx.gif)
